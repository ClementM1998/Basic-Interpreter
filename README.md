Cara menggunakan:

* Untuk Windows
 1. Muat turun 'Basic.jar' dan 'Basic.cmd'
 2. Jalankan Basic.cmd

* Untuk Linux
1. Muat turun 'Basic.jar' dan 'Basic.sh'
2. Jadikan fail 'Basic.sh' boleh dijalankan
   chmod +x Basic.sh
3. Jalankan Basic.sh
